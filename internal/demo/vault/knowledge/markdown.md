# Markdown Syntax

LazyMD is a markdown-native editor. Here's a quick syntax reference.

## Text Formatting

- **Bold** with `**text**`
- *Italic* with `*text*`
- ~~Strikethrough~~ with `~~text~~`
- `Code` with backticks

## Block Elements

> Blockquotes use `>` prefix

- Unordered lists with `-`
1. Ordered lists with numbers

## Task Lists

- [x] Completed task
- [ ] Pending task

## Links

- Standard: [text](url)
- Wiki-link: [[note-name]]
