# Navigation

| Key | Action |
|-----|--------|
| <kbd>h</kbd> <kbd>j</kbd> <kbd>k</kbd> <kbd>l</kbd> | Move cursor left / down / up / right |
| <kbd>w</kbd> | Jump to next word |
| <kbd>b</kbd> | Jump to previous word |
| <kbd>e</kbd> | Jump to end of word |
| <kbd>0</kbd> | Go to beginning of line |
| <kbd>$</kbd> | Go to end of line |
| <kbd>^</kbd> | Go to first non-blank character |
| <kbd>g</kbd> | Go to first line |
| <kbd>G</kbd> | Go to last line |
| <kbd>Ctrl+D</kbd> | Scroll half page down |
| <kbd>Ctrl+U</kbd> | Scroll half page up |
| <kbd>Page Up</kbd> / <kbd>Page Down</kbd> | Scroll full page |

Most motions accept a count prefix. Type a number before the motion to repeat it (e.g., <kbd>5j</kbd> moves down 5 lines).
